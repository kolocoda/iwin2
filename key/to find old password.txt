just search for the file idea.log in the directory C:\Users\%UserName%\.AndroidStudio1.5\system\log

here you will find atleast 7 or 8 files naming - idea.log1 , idea.log2 and so on. In one of the file you will get the alias name password.
Something like this,
-Pandroid.injected.signing.store.file=/Users/myuserid/AndroidStudioProjects/keystore/appkey.jks,
-Pandroid.injected.signing.store.password=mystorepassword,
-Pandroid.injected.signing.key.alias=myandroidkey,
-Pandroid.injected.signing.key.password=mykeypassword,

Note : this will work iff you have released app atleast once.
It worked for me though. It doesn't need any tool. Its just a trick.?

to change password
keytool -storepasswd -new [insert new keystore password] -keystore [insert keystore file name]

keytool -keypasswd -alias [insert alias] -new [insert new key password] -keystore [insert keystore file name]
